## Tool Functionality
This tool takes a paragraph, summarizes it into structured and concise bullet point formed information.

## Prerequisites
- Node.js
- Gemini API key

## Setup Steps
- Clone the project
- Install dependencies by running "npm install" in the terminal
- Create an .env file, and store your Gemini AI API key in a variable called "API_KEY"
- In the terminal run "node index.js"
